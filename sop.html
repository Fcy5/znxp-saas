选品流程 — 七维综合评估体系

所有维度数据全部呈现，AI综合加权打分，数据说话不靠感觉

📌
核心原则：不能仅靠AI评分决策——必须查看每一个维度的原始数据，理解数据背后的逻辑，再结合AI综合评分做判断。利润维度一票否决制：任何商品若ROI预期 < 120%，无论其他维度多高，直接否决。
七大评估维度 · 权重与评分标准
维度 权重 核心指标（必须全部查看原始数据） 满分标准
① 广告验证 25% FB在投广告主数量 / 最长在投时长 / 单条互动量(点赞+评论+分享) / 广告视频完播率 广告主≥5 / 在投≥21天 / 互动≥500 / 完播率>20%
② 社媒热度 20% TikTok 7天相关视频总播放量 / 点赞≥500的小红书笔记数 / Instagram话题帖量 / YouTube评测数 TikTok总播≥500万 / 小红书≥10条 / IG话题≥5000帖

⑥ 趋势时效 5% Google Trends 12个月曲线(截图存档) / 是否季节性产品 / 当前处于趋势何位置 趋势上升或平稳 / 非临近旺季末尾 / 非夕阳品类
⑦ 受众匹配 5% 目标人群画像清晰度 / 与现有店铺客群重叠度 / 复购周期 / 送礼属性强弱 画像明确 / 与现有客群重叠≥60% / 有复购潜力

选品流程 — 操作执行步骤

每一步必须完成，不允许跳步，完成后在检查表打勾存档

1
FB广告库初筛（每日09:30开始，30分钟内完成）
进入 facebook.com/ads/library，按品类关键词搜索（如 custom gift / personalized / couple）
筛选条件：国家选美国/英国/澳洲，广告类型选全部，记录以下数据并截图存档：
· 广告主名称 + 官网域名（逆查Shopify商品结构）
· 在投时长（日期范围）：低于14天不记录
· 互动量（点赞+评论+分享）：低于200不记录
· 广告形式（视频/图文/轮播）：视频广告优先
每日目标：记录5-8个候选广告主，填入「选品候选记录表」
2
TikTok + 小红书热度交叉验证
TikTok Creative Center搜索同款关键词：
· 筛选近7天视频，播放量排序，记录TOP5的播放量总和
· 注意评论区是否有购买意向留言（"where to buy" / "link?"）
小红书搜索同款：点赞≥500笔记数量，评论中是否有购买询问
两平台都有热度 → 优先级自动上升一档
3
供应链询价（利润维度，一票否决关键步骤）
在1688搜索同款，联系≥3家供应商询价，重点问：
· 1件起批价格（不要问MOQ）
· 定制/印刷附加费
· 发货周期（注意：>7天生产期需要提示客户）
计算总成本 = 供货价 + 尾程运费 + 包装$0.5-2 + 支付手续费(售价×3%)
若任何供应商方案ROI < 120%，本商品直接停止评估
4
平台录入 + AI七维综合打分 AI 必做
在ZNXP平台「选品大厅」找到对应商品，或手动新增，填写所有采集到的原始数据：
fb_ad_count / fb_ad_max_days / fb_ad_max_engagement / tiktok_views_7d / 供货价 / 竞品售价区间
点击「AI综合评分」，系统按七维模型打分，必须查看每个维度的分项得分，不能只看总分
将评分截图 + 原始数据一并存入选品候选记录表
5
Google Trends趋势核查
在Google Trends输入商品核心关键词，查看：
· 过去12个月趋势（截图，标注当前所在趋势位置）
· 相关搜索词（发现长尾词机会）
· 地区热度分布（确认目标市场是否有需求）
季节性商品：确认当前时间距旺季还有≥6周，否则推迟到下个周期
6
加入选品库 + 周会评审
满足条件（综合分≥75 + 利润分≥60）→ 点击「加入我的选品库」，备注：
来源渠道 / 竞品均价 / 预期定价 / 目标受众一句话描述 / 供应商联系方式
每周一选品会议：团队评审本周所有候选，投票选出5-8件进入「本周测品名单」
同时测品上限：≤ 12件（超过此数广告预算分散，数据无效）
7
测品 & 淘汰机制（上架后执行）
【前置：测品有效性判定】
以下三个流量渠道，至少满足 2项，才算进入有效测品计时；否则补足后重新计时，不计入淘汰判断：
  ① 付费广告：已投放广告预算 ≥ $30，且广告正常运行 ≥ 3天
  ② 自然流量：商品页UV ≥ 100（7天累计，来源：SEO / 直接访问 / 邮件）
  ③ 社媒发布：TikTok / Instagram / Facebook 已发布 ≥ 2条含商品链接内容

【第一阶段：7天生死线】
· 测品预算：每件 $50–100 广告费，周期7天
· 7天0单，且已满足「有效测品」前置条件 → 立即下架，按以下6个维度归档失败原因：
  - 价格：加购量高但付款率低（加购未付款率 > 30%）→ 定价可能偏高
  - 图片 / 落地页：广告展示量高但点击率 CTR < 1% → 主图或文案吸引力不足
  - 受众：广告 CPM 高、频次低、点击极少 → 受众定向有偏差
  - 平台差异：分别记录 FB / TikTok / Google 各渠道 CTR 与 ROAS，找出最差短板
  - 社媒内容：帖子平均互动率 < 1% → 内容方向与目标人群不匹配
  - 时机：上架时间是否错过节日 / 季节窗口期（如情人节品在2月5日后上架）
· 7天0单，但「有效测品」未满足（未投广告 + 未发社媒 + 自然UV极低）→ 不算有效失败，补足流量后重新计时7天

【第二阶段：14天 ROAS 预警 → 进入优化期】
· 触发条件A（已投广告）：上架14天，累计广告花费 ≥ $80，且 ROAS < 1.5
· 触发条件B（未投广告，依赖自然流量 / 社媒）：上架14天，订单数 < 3单
· 进入优化期后，须在 48小时内 完成以下至少 2项 干预措施，并记录干预时间和具体改动：
  - 主图 A/B 测试：新旧图各跑3天，对比 CTR 变化
  - 重写广告文案 / 商品描述：对标竞品 Top 广告的语言风格
  - 调整定价：降价 5–15% 测转化率，或小幅升价 测高端受众接受度
  - 重置广告受众：扩宽兴趣词，或切换 LAL 相似受众
  - 补充社媒内容：额外发布 ≥ 3条不同角度内容（开箱 / 穿搭对比 / 使用场景）

【第三阶段：优化后14天最终裁决 → 永久淘汰 或 留存】
· 优化后14天：ROAS 仍 < 1.5（投广告场景），或订单无明显增长（自然/社媒场景）→ 永久淘汰
· 永久淘汰商品进入「避坑库」，必须完整填写以下字段方可归档：
  - 商品ID / 品名 / 上架日期 / 下架日期
  - 累计广告花费 / 累计订单数 / 最终 ROAS（无广告填 N/A）
  - 流量渠道覆盖情况：广告 ✓/✗   自然流量 ✓/✗   社媒发布 ✓/✗
  - 失败主因（从第一阶段6个维度中选1–2个，写具体数据支撑）
  - 优化措施记录：做了什么 → 结果数据如何变化
  - 是否可再激活标记：季节性（标记旺季窗口）/ 换平台测试 / 换受众后重试
⚠️
禁止行为：禁止凭感觉或"好看"加入选品库 ｜ 禁止跳过供应商询价直接上架 ｜ 禁止同时测品超过12件 ｜ 禁止在季节性商品旺季最后4周才开始备货测品

1
商品导入 Shopify（草稿状态，禁止直接上架）
通过「改款上架」功能推送至目标店铺，初始状态强制设为Draft（草稿）
必须填写完整：SKU标识 / 尺寸变体 / 重量（运费计算用）/ 供货商信息
主图要求：≥ 1000×1000px，纯白底或生活场景，无水印，无竞品Logo，≥ 3张
每次导入商品数量：≤ 10件/天（保证整改质量）
2
AI SEO 优化 AI 必做
进入「Shopify AI优化」→ 选择店铺 → 勾选商品 → 一键SEO优化
AI自动生成：SEO标题（含核心关键词，≤70字符）/ Meta描述（含关键词，≤155字符）/ 图片Alt标签（每张不同）/ 结构化数据JSON-LD
人工审核要点：关键词是否自然融入（非堆砌）/ 标题是否有吸引力 / 描述是否包含CTA
审核通过后点击「应用优化」，写入Shopify
3
商品描述整改标准
结构（缺一不可）：① 痛点引入（情感共鸣，1-2句）→ ② 产品如何解决问题 → ③ 核心卖点3-5条（加粗）→ ④ 规格参数（材质/尺寸/重量）→ ⑤ 定制说明（如有）→ ⑥ 配送时效
字数：英文≥300词 / 中文≥200字，禁止复制竞品原文
禁用词：best / cheapest / guaranteed / cure / #1（触发广告政策违规）
加分项：嵌入场景化使用图片 / 添加FAQ（减少客服压力）
4
素材生产 — AI图生视频 AI 必做
进入商品详情Drawer → 底部「AI视频 & 社媒素材」→ 选择主图 → 点击生成
等待60-120秒，生成3-8秒展示视频（wan2.7-i2v模型）
用途规划：TikTok有机流量 / Facebook视频广告 / Instagram Reels / GMC购物广告
下载后按「品牌-SKU-日期」命名存入素材库，不要重复生成相同主图
5
ROI定价公式（此步骤完成前禁止上架）
总成本计算：供货价 + 尾程运费 + 包装费($0.5-2) + 支付手续费(售价×3%) + 预估广告成本/件($2-5)
最低定价 = 总成本 × 2.2（保证ROI ≥ 120%）
推荐定价 = 总成本 × 2.5-3.5（ROI 150%-250%，为广告优化留空间）
心理定价：取推荐定价后，调整为$X4.99 / $X9.99格式
对比价（Compare at Price）= 推荐定价 × 1.3-1.5，制造折扣感
批量调价使用平台「批量改价」功能，禁止手动逐一修改
📐 ROI定价快速参照表
最低售价 = 总成本 × 2.2 ｜ 推荐售价 = 总成本 × 3.0
总成本$8 → 最低$17.6 → 推荐$24 → 定价$23.99 ｜ 总成本$12 → 最低$26.4 → 推荐$36 → 定价$35.99 ｜ 总成本$18 → 最低$39.6 → 推荐$54 → 定价$54.99 ｜ 总成本$25 → 最低$55 → 推荐$75 → 定价$74.99
6
上架前最终检查清单（必须全部勾选才能切换状态为Active）
  • 主图≥3张，无水印，无竞品Logo，最大边≥1000px
  • SEO标题已优化，含核心关键词，≤70字符
  • Meta描述已优化，含关键词和CTA，≤155字符
  • 商品描述≥300词，含痛点+卖点+规格+配送时效
  • 价格已按ROI公式设定，含Compare at Price
  • 库存设置正确（或勾选无限库存）
  • 标签(Tags)已填写（品类+受众+节日关键词）
  • AI视频素材已生成并存档
7
上架后立即推送 — GMC + 社媒
推送GMC：进入「Google购物广告」→ 选择店铺 → 勾选刚上架商品 → 推送
GMC审核周期24-72小时，期间同步进行社媒推广
推送社媒：使用AI生成的视频素材，按第04章《社媒发布流程》在TikTok/Facebook/Instagram同步发布新品帖
上架后第1周：每天监控订单数据，若3天0单立即排查（价格/主图/受众定向问题）

独立站 AI 诊断流程

将完整独立站喂给AI，逐页检查布局排版与转化障碍，每月至少执行一次

🔍
执行时机：新店上线前必做 / 每月第一个周一执行 / 广告ROAS突然下降20%以上时立即执行。诊断结果需形成书面优化清单，按优先级在2周内完成整改。
第一步 — 截图准备（将以下页面全截图，发给AI）
需要截图的页面 截图要求 截图工具
首页（Homepage）全页滚动截图，桌面端 + 手机端各一张GoFullPage插件 / Chrome截图
商品列表页（Collection）最热门品类页，含筛选栏同上
商品详情页（PDP）选最畅销商品，全页截图同上
购物车页（Cart）含至少1件商品的状态同上
结账页（Checkout）手机端截图（70%+用户用手机结账）Chrome手机模拟器
404页 / 关于我们页检查品牌信任度元素同上
第二步 — AI诊断提示词模板 AI 必做
💬
发给AI的Prompt（逐页使用）：
「你是一位资深跨境电商独立站转化优化专家。我将给你一张我的Shopify独立站【页面名称】的截图。请从以下维度逐一分析并给出具体优化建议：
1. 视觉层次与信息架构（用户是否能在3秒内找到最重要内容）
2. CTA按钮的位置、颜色、文字是否足够突出（是否被忽视）
3. 字体大小与可读性（手机端是否≥14px）
4. 白空间使用（是否拥挤/空旷）
5. 信任元素（评价/安全支付/退换货政策是否可见）
6. 图片质量与加载速度预判
7. 移动端适配问题（按钮是否好点击，表单是否方便填写）
8. 转化障碍点（哪里最可能导致用户离开）
请列出TOP3最紧急需要修复的问题，并给出具体操作建议。」
第三步 — 常见问题快速修复指南
问题类型 症状 快速修复方案 优先级
首页英雄区无吸引力 Banner图文字看不清 / 无明确CTA 更换高对比度Banner，加大标题字号，CTA按钮改为亮色（橙/绿） P0
手机端按钮太小 拇指难以点击，加购失败 Add to Cart按钮高度改为≥52px，字体≥16px，颜色强对比 P0
无信任元素 用户不敢下单 在页脚/购物车页加：⭐评分展示 / 🔒安全支付图标 / 退款保障文字 P1
页面加载慢 图片过大，跳出率高 所有商品图压缩至≤300KB，使用WebP格式，启用Shopify CDN P1
结账流程过长 填写步骤多，弃单率高 启用Shopify一页结账 / 添加Shop Pay / PayPal快捷支付 P1
商品描述无结构 大段文字无格式，用户不读 改为图标+短句列表，关键词加粗，添加产品规格表格 P2
第四步 — 整改追踪
✓
