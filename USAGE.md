## 1. 注册与登录

### 注册账号

1. 打开平台网址，点击**注册**
2. 输入您的**邮箱地址**，点击**发送验证码**
3. 查收邮件，将 6 位验证码填入表单
4. 填写**用户名**和**登录密码**
5. 点击**完成注册**，自动进入平台

### 登录

1. 输入注册时使用的邮箱和密码
2. 点击**登录**即可进入

> **提示：** 登录状态会在一段时间后失效，失效后系统会自动跳转到登录页，重新登录即可。

## 2. 数据看板

登录后默认进入数据看板页面，一眼掌握您的运营数据。

### 顶部统计卡片

| 卡片 | 含义 |
|------|------|
| 今日推品数 | 平台今天新抓取入库的商品数量 |
| 选品库商品数 | 您收藏进选品库的商品总数 |
| 今日上架数 | 今天推送到 Shopify 的商品数 |
| 总上架数 | 累计成功上架的商品总数 |
| 平台商品总量 | 整个平台商品池的商品数量 |
| AI 任务运行中 | 当前正在后台执行的 AI 任务数 |
| AI 任务今日完成 | 今天已完成的 AI 分析任务数 |

### AI 今日推品

看板下方展示 **20 条**平台每日精选推荐商品，由 AI 综合以下维度评分筛选：

- 销售趋势热度
- AI 综合评分
- 利润空间估算
- 社媒（TikTok）播放量
- Facebook 广告密度
- 买家评分

点击任意商品卡片可进入详情页，查看完整数据并进行 AI 改款。

## 3. 选品大厅

路径：顶部导航 → **选品大厅**

这里是平台全量商品池，当前收录 **16,000+ 条**来自 TikTok、Facebook、Google、Shopify 的商品数据。

### 搜索与筛选

页面顶部提供多维度筛选条件：

- **关键词**：按商品标题搜索
- **品类**：从下拉菜单选择品类（服装、家居、3C等）
- **来源平台**：TikTok / Facebook / Google / Shopify
- **品牌**：快速筛选知名跨境品牌
- **价格区间**：设置最低/最高价（美元）
- **利润空间**：筛选高利润率商品
- **销售趋势**：筛选销售趋势分达标的商品
- **排序**：按 AI 评分、价格、趋势、入库时间排序，支持升序/降序

设置好筛选条件后点击**搜索**，或直接修改筛选项自动刷新结果。

### 单件收藏

点击商品卡片右上角的**收藏图标（书签）**，即可将商品加入选品库，方便后续查看和上架。

### 批量收藏

当您想一次性收藏多件商品时：

1. 点击页面右上角的**多选**按钮，进入批量模式
2. 商品卡片左上角出现复选框，点击卡片即可选中（再次点击取消）
3. 页面底部弹出操作栏：
   - **全选本页**：选中当前页所有商品
   - **清空**：取消所有选中
   - **批量加入选品库**：一键将选中商品全部收藏（最多 100 件/次，已收藏的自动跳过）
4. 点击操作栏右侧 **X** 退出批量模式

## 4. 商品详情与 AI 改款

在选品大厅或数据看板点击任意商品，进入商品详情页。

### 商品数据

左侧展示完整商品信息：
- 主图及多张副图
- 标题、原始描述、定价、品类、品牌
- 多维数据指标：TikTok 播放量、Facebook 广告数、GMC 搜索量、Etsy 收藏量、评分及评价数
- AI 综合评分（满分 100）、估算利润空间

### AI 生成文案

1. 在右侧面板选择**语言**（中文 / 英文）
2. 选择 **AI 模型**（推荐 Gemini 2.5 Flash，速度快；Claude Sonnet 4.6 质量高）
3. 点击**生成文案**，稍等片刻自动填充：
   - SEO 标题
   - Meta 描述
   - 商品 HTML 描述（适配 Shopify）
   - 图片 Alt Tags
4. 可手动修改后用于上架

### AI 生成图片

1. 选择**图片模型**（推荐 GPT-Image 1.5）
2. 选择**预设提示词**，或自行输入：
   - 白底产品图
   - 欧美生活场景
   - 礼品展示
   - 去除水印重绘
   - 节日主题
   - 欧美模特展示
   - 模特街拍
3. （可选）上传**参考图**，实现以图改图
4. 点击**生成图片**，生成完毕后可将图片加入上架图片列表

### AI 生成社媒文案

点击**生成社媒文案**，自动输出适合以下平台的推广文案：
- TikTok 脚本
- Facebook 广告文案
- Instagram 说明文字

每条文案旁有**一键复制**按钮，直接粘贴到对应平台使用。

## 6. 绑定 Shopify 店铺

路径：顶部导航 → **店铺管理**

使用改款上架功能前，需先绑定您的 Shopify 店铺。

### 绑定步骤

1. 点击页面右上角**绑定新店铺**（或点击虚线卡片）
2. 填写以下信息：
   - **店铺域名**：格式为 `yourstore.myshopify.com`（不含 https://）
   - **店铺名称**：自定义，用于平台内显示
   - **Admin API Access Token**：在 Shopify 后台获取（见下方说明）
3. 点击**确认绑定**

### 如何获取 Admin API Access Token

1. 登录 Shopify 后台（`admin.shopify.com`）
2. 左侧菜单 → **Apps** → **App and sales channel settings**
3. 点击**Develop apps** → **Create an app**
4. 创建应用后，进入 **Configuration** → 开启所需权限（至少需要 `write_products`）
5. 点击 **Install app** 安装
6. 进入 **API credentials** 标签页，复制 **Admin API access token**（格式：`shpat_xxxxx`）

> **注意：** Token 只显示一次，请妥善保存。

### 绑定多个店铺

支持绑定多个 Shopify 店铺，每个店铺独立显示为一张卡片，可分别进行 AI 诊脉、推品和上架操作。

### 解绑店铺

在店铺卡片底部点击**解绑店铺**，确认后移除该店铺（已上架商品记录不受影响）。

## 7. AI 店铺诊脉与智能推品

在店铺管理页，每张店铺卡片提供两个 AI 功能：

### AI 店铺诊脉

> 分析您的店铺定位、视觉风格和目标受众，生成结构化店铺画像。

**使用步骤：**
1. 在对应店铺卡片点击**AI 店铺诊脉**
2. 系统自动在后台分析您的店铺（通常需要 30 秒至 2 分钟）
3. 任务卡片实时显示当前进度
4. 完成后，点击任务卡片的**展开箭头**查看分析结果：
   - **定位**：您的店铺所在细分市场（如：定制刺绣服装）
   - **风格**：视觉风格描述（如：温馨家庭风、简约北欧风）
   - **画像摘要**：整体描述
5. 画像生成后，常驻显示在店铺卡片上，后续智能推品会基于此画像进行匹配

### 智能推品

> 基于店铺画像，从平台 16,000+ 件商品中自动筛选最适合您店铺风格的商品。

**使用步骤：**
1. 建议先完成**AI 店铺诊脉**（有画像则推品更精准）
2. 点击**智能推品**，系统后台自动执行
3. 完成后展开任务卡片，查看推荐的 10 件商品
4. 点击商品标题，直接跳转至该商品详情页，可进行 AI 改款和上架

### 任务状态说明

| 状态 | 说明 |
|------|------|
| 排队中 | 任务已提交，等待执行 |
| 执行中 | 正在分析，显示进度条 |
| 完成 | 分析成功，可展开查看结果 |
| 失败 | 执行失败，显示失败原因 |

- 完成或失败的任务可点击右上角 **X** 关闭任务卡片
- 页面内可同时运行多个任务

## 8. 改款上架

在商品详情页，右侧面板完成内容修改后，向下滚动至**上架设置**区域：

### 上架步骤

1. **选择目标店铺**：从已绑定的店铺中选择
2. **修改上架内容**（所有项目均可选填，不填则使用原始数据）：
   - **标题**：修改商品标题（建议使用 AI 生成的 SEO 标题）
   - **描述**：商品详情（支持 HTML，可直接使用 AI 生成的描述）
   - **定价**：设置您的售价（美元）
   - **商品类型**：Shopify 分类标签
   - **Tags**：搜索标签，用英文逗号分隔
3. **添加尺寸变体**（适用于服装类商品）：
   - 点击**添加尺寸**，为每个 Size 单独设置价格
   - 支持添加多个尺寸（S/M/L/XL 等）
4. **上架图片**：
   - 默认使用商品原始主图
   - 可上传本地图片或选择 AI 生成的图片作为附加图
5. 点击**上架到 Shopify**，等待几秒钟

### 上架结果

- **成功**：商品已在 Shopify 店铺上线，可点击弹出的 Shopify 链接直接查看
- **失败**：页面显示失败原因（常见原因：Token 权限不足、店铺域名错误）

## 9. 上架历史

路径：顶部导航 → **上架历史**

查看所有历史上架记录，支持按店铺筛选。

### 商品状态

| 状态 | 含义 |
|------|------|
| 已上架 | 商品在 Shopify 正常在售 |
| 处理中 | 上架请求正在处理 |
| 失败 | 上架失败（可查看失败原因） |
| 已下架 | 商品已从 Shopify 下架 |

### 行内操作

**已上架商品可以：**

- **点击 Shopify 链接**：跳转到 Shopify 店铺中的商品页面
- **改价**：点击**改价**按钮，弹出输入框，填写新价格（美元）后确认，自动同步更新 Shopify 上所有变体价格
- **下架**：点击**下架**按钮，确认后将该商品在 Shopify 下架（状态变为 archived），记录从列表移除

**失败或已下架商品可以：**

- **重新上架**：以相同的标题、价格重新提交上架请求

### 筛选与翻页

- 顶部店铺筛选栏：点击**全部店铺**或具体店铺名称，过滤对应记录
- 底部翻页：每页展示 20 条记录，支持上一页/下一页

## 10. Facebook 广告资料库

路径：顶部导航 → **广告资料库**

收录了平台从 Facebook 广告库抓取的竞品广告数据（**192+ 条**），帮助您分析竞品投放策略。

### 功能

- **关键词搜索**：按广告标题或文案内容搜索
- **浏览广告**：查看广告主图、广告文案、投放数据
- 所有广告图片均已保存至本地服务器，不会因链接过期而无法查看

### 使用建议

- 分析竞品的广告卖点和文案风格
- 挖掘市场热门品类和选品灵感
- 结合平台的选品数据，找到有广告验证的潜力商品

## 11. 供应商管理

路径：顶部导航 → **供应商**

用于管理您的货源供应商及其商品库。

### 添加供应商

1. 点击**添加供应商**
2. 填写供应商名称、Logo 图片链接、联系电话/邮箱、简介
3. 点击确认保存

### 管理供应商商品

在供应商卡片中：

1. 点击**查看商品**进入该供应商的商品列表
2. 点击**添加商品**录入新商品：
   - 标题、品类、价格、商品图片、商品链接
3. **上架/下架**：点击切换商品状态（上架=在售，下架=暂停）
4. **删除**：从供应商商品库中移除该商品

### 一键上架供应商商品

在供应商商品列表中，点击商品的**上架**按钮，进入改款上架流程，将该供应商商品直接推送至您的 Shopify 店铺。

## 12. 系统设置

路径：右上角头像 → **设置**，或侧边栏底部**设置**入口

### 账号信息

- 查看当前**用户名**和**邮箱**
- 修改用户名
- 修改邮箱地址
- 修改登录密码（需先输入当前密码验证身份）

修改完成后点击**保存修改**。

### AI 模型配置

- **默认文案模型**：选择生成商品文案和社媒文案时默认使用的 AI 模型
- **默认图片模型**：选择 AI 图片生成时默认使用的模型

建议配置：
- 文案：**Gemini 2.5 Flash**（速度与质量均衡）或 **Claude Sonnet 4.6**（中文表达更自然）
- 图片：**GPT-Image 1.5**（效果最佳）

设置保存后，在商品详情页打开 AI 功能时会自动选中您配置的默认模型。

## 常见问题

**Q：上架失败，提示权限不足怎么办？**  
A：检查 Shopify Admin API Token 是否开启了 `write_products` 权限。进入 Shopify 后台 → Apps → 找到对应自定义应用 → Configuration → 勾选所需权限后重新安装，获取新 Token 重新绑定店铺。

**Q：AI 生图/文案速度很慢？**  
A：切换到更快的模型（如 Gemini 2.5 Flash Lite），或在非高峰时段使用。

**Q：智能推品结果不准确？**  
A：先执行**AI 店铺诊脉**，让系统了解您的店铺定位，再使用智能推品，结果会更精准。

**Q：改价后 Shopify 上的价格没有变化？**  
A：价格更新需要几分钟生效，刷新 Shopify 后台页面查看。如仍未更新，检查店铺 Token 是否有 `write_products` 权限。

**Q：批量收藏时提示"已存在"是什么意思？**  
A：选中的商品中有部分已经在您的选品库中，系统会自动跳过已收藏的，其余商品正常加入。
