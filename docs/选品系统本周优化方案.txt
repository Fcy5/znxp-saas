    <section class="page">
      <div class="section-kicker">Automation Map</div>
      <h2>自动化选品的 6 组核心能力</h2>
      <p class="lead">自动化选品的核心，不是简单把人工判断搬到页面上，而是让系统持续收集信号、自动分层、自动筛候选、自动输出推荐解释。这 6 组能力共同组成自动化选品的核心结构。</p>

      <div class="auto-grid">
        <div class="auto-card">
          <h3>1. 自动打标签</h3>
          <p>现在标签需要人来补，自动化选品要求系统先自动判断，再允许人工修正。</p>
          <ul>
            <li>自动识别节日、季节、人群、场景</li>
            <li>根据标题、图片、平台来源和内容信号自动归类</li>
            <li>先机判，后人工微调</li>
          </ul>
        </div>
        <div class="auto-card">
          <h3>2. 自动候选生成</h3>
          <p>候选池生成目标是从全量商品池中按规则每日自动产出候选结果。</p>
          <ul>
            <li>按平台和品类自动拉新</li>
            <li>自动按周专题池分配候选</li>
            <li>自动淘汰低相关度商品</li>
          </ul>
        </div>
        <div class="auto-card">
          <h3>3. 自动刺绣适配判断</h3>
          <p>这是定制刺绣系统特有的一层，不做这一层，自动化也只是通用选品。</p>
          <ul>
            <li>自动判断刺绣位是否明显</li>
            <li>自动判断是否适合名字、称呼、日期、线稿</li>
            <li>自动判断是否适合礼品表达和成片展示</li>
          </ul>
        </div>
        <div class="auto-card">
          <h3>4. 自动爆款评分</h3>
          <p>现在的推荐分更像热度分，自动化选品要升级成刺绣爆款分。</p>
          <ul>
            <li>平台热度 + 内容热度 + 广告热度</li>
            <li>节日匹配度 + 礼品属性 + 刺绣适配度</li>
            <li>按专题池分别算分，形成分专题的评估体系</li>
          </ul>
        </div>
        <div class="auto-card">
          <h3>5. 自动解释为什么推</h3>
          <p>自动化选品不应只输出分数，系统还需要提供可读的推荐理由，便于复核和追溯。</p>
          <ul>
            <li>自动生成选品理由</li>
            <li>自动说明适合什么节日、什么人群、什么内容角度</li>
            <li>能追溯推这个商品的依据</li>
          </ul>
        </div>
        <div class="auto-card">
          <h3>6. 自动复盘和反馈</h3>
          <p>没有反馈闭环，自动化只会一直重复同样的错误。</p>
          <ul>
            <li>记录主推、淘汰、误判原因</li>
            <li>自动修正下一轮专题池策略</li>
            <li>形成“越选越准”的闭环</li>
          </ul>
        </div>
      </div>

      <div class="footer-note">
        <strong>自动化选品的真实目标</strong>
        <p>最终目标是让系统每天自动给出一批值得关注的候选款、自动解释推荐理由、自动沉淀重点款，人工只做复核和拍板。</p>
      </div>
    </section>

      <h2>商品维度必须扩充，不然选品永远不够深</h2>
      <p class="lead">当前商品维度更偏平台通用数据，比如热度、评价、利润、广告数。极致选品系统还需要补齐“是否适合做定制刺绣”和“是否适合进入当前专题池”的维度，才能从热度判断进一步提升到适配度判断。</p>

            <li>holiday_tags</li>

            <li>event_window：预热期 / 爆发期 / 尾期</li>
          </ul>
        </div>
        <div class="dim-card">

            <li>scenario_tags</li>
            <li>gift_relation：爸爸 / 母亲 / 宠物 / 毕业 / 家庭</li>
            <li>usage_scene：旅行 / bbq / 居家 / 纪念</li>
          </ul>
        </div>
        <div class="dim-card">
          <h3>3. 刺绣适配维度</h3>
          <p>这是定制刺绣系统和普通选品系统拉开差距的关键。</p>
          <ul>
            <li>embroidery_position：帽前 / 胸前 / 袖口 / 包面</li>
            <li>customization_type：名字 / 日期 / 称呼 / 线稿</li>
            <li>embroidery_visibility：刺绣识别度</li>
            <li>giftability：礼物感强弱</li>
          </ul>
        </div>

          <p>让系统知道这件商品适不适合做内容表达。</p>
          <ul>
            <li>content_hook：内容切入口</li>
            <li>visual_impact：首图冲击力</li>
            <li>video_potential：短视频展示潜力</li>
            <li>ugc_potential：用户自发分享潜力</li>
          </ul>
        </div>

          <ul>
            <li>selection_status</li>
            <li>selection_reason</li>
            <li>selection_confidence：判断信心</li>
            <li>manual_review_flag：是否需要人工复核</li>
          </ul>
        </div>
        <div class="dim-card">
          <h3>6. 自动化分数维度</h3>
          <p>最终要把多个信号压缩成可排序、可复核的自动选品分。</p>
          <ul>
            <li>trend_score：趋势热度分</li>

            <li>gift_score：礼品属性分</li>
            <li>campaign_score：专题匹配分</li>
          </ul>
        </div>
      </div>
